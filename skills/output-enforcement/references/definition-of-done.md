# Definition of Done

The standing bar every Conversion Skills deliverable clears before anyone calls it done. Run it as the last check. If a line fails, the work is not done, fix it or cut the offending part.

- [ ] Every external claim cites a source or is cut. No unsourced fact survives.
- [ ] Zero em-dashes and zero banned brand words. Mechanical scan, not a vibe read.
- [ ] One clear next action or CTA on anything outbound. The reader knows what to do next.
- [ ] A ledger row appended if a metric moved. Append-only to `Memory/kpi-ledger.md`, never edit a prior row.
- [ ] Draft-only respected. Nothing sent, posted, or published. Outbound stays a draft for a human to release.
- [ ] Agency firewall respected. Only the active `Clients/{slug}/` was read, never a sibling client.
- [ ] Universal frontmatter complete. Name matches the folder, description is one line with no colon, no newline, no em-dash, no banned word.
- [ ] No invented numbers or client names. Every figure traces to a real source, every name is real.
