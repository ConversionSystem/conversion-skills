# Sales Page Spec

A build spec for the Conversion Skills page. Conversion System brand. Inter Tight, JetBrains Mono for numbers, the six colors, Orange used twice at most. No em-dashes. See `BRAND.md` and `assets/brand-tokens.css`.

## Nav
The wordmark `CONVERSION / SYSTEM` with `an AI agency`. Links: Product, Pricing, Docs, GitHub. One Orange button: "Get Conversion Skills".

## Hero
- Headline (Inter Tight, large): "Your tools forget. Conversion Skills don't."
- Subhead (one line): "99 Claude skills that audit, prospect, write, and brief your business, and chart what moved each month. Plain markdown you own. No database."
- Two buttons: Orange "Get Conversion Skills", Ink outline "Read the docs".

## Receipts band (Ink background, JetBrains Mono numbers)
Four receipts. Each a number, a context, a named source where you have one.
- `44` skills, one install.
- `5` gates, green on every push.
- `10 min` from empty folder to a working vault.
- `0` cross-client leaks, proven by the firewall test.

## What it does
The 8 modules, one line each (Setup, Memory, Operator, Optimizer, Team OS, Client OS, Connector layer, Workflow Library). Pull from `README.md`.

## The moat
One short block on the append-only ledger. The visual: a ledger row growing month over month, the trend line going up. Headline: "It gets cheaper to grow every month, because the memory compounds."

## Comparison row
| Trait | A folder of notes | A CRM | Conversion Skills |

| Does the work | No | No | 44 skills |

## Pricing
The five tiers from `docs/pricing.md`. Fixed, named, refundable.
